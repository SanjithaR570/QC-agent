Quality Control (QC) Agent
📌 Project Description

This project is a simple Quality Control (QC) Agent developed using Python.
The agent automatically generates quality control procedures for new product lines based on user input.

🎯 Problem Statement

Develop an agent that generates quality control procedures for new product lines.

🚀 Features
Takes product details as input
Supports multiple product types (food, electronic, medicine)
Generates QC procedures automatically
Includes:
Packaging and labeling checks
Safety checks
Product-specific tests
Material-based checks
Fragile product handling
🛠️ Technologies Used
Python
Visual Studio Code

▶️ How to Run the Project
Install Python
Open the project in VS Code
Open terminal

Run the command:

python qc_agent.py
Enter product details when prompted
🧪 Sample Input
Product Name: Mango Juice  
Product Type: food  
Material: mango  
Manufacturing Date: 01-04-2026  
Expiry Date: 01-06-2026  
Fragile: no  
📋 Sample Output
QC Procedures for Mango Juice:
1. Check packaging integrity
2. Verify labeling accuracy
3. Check taste and freshness
4. Test for microbial safety
5. Verify expiry date
6. Check quality of material: mango
💡 How It Works

The agent follows this process:

Input → Processing (Logic) → Output

Input: Product details
Processing: Applies predefined rules
Output: QC procedures
